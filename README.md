# MNIST Handwritten Digit Recognizer

A complete Convolutional Neural Network (CNN) implementation for recognizing handwritten digits from the MNIST dataset.

## Features

- **CNN Architecture**: 3 convolutional layers with max pooling and dropout
- **Data Augmentation**: Random rotation and translation for better generalization
- **Training**: Adam optimizer with learning rate scheduling
- **Evaluation**: Confusion matrix, classification report, and sample visualizations
- **Prediction**: Support for custom images
- **GPU Support**: Automatic detection and utilization
